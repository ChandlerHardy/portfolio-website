# Portfolio Website — Product Context

## What
Chandler Hardy personal portfolio. chandlerOS aesthetic (Win95 desktop metaphor) showcasing projects and skills.

## Target Users
Hiring managers, recruiters, and potential collaborators evaluating Chandler's work.

## Current State
- chandlerOS (Win95 desktop) redesign on feature/portfolio-v2
- 3 showcase projects: Crooked Finger, Elucidate Chess, GreenLine
- Next.js 15, React 19, Tailwind, Motion (Framer Motion)
- Deployed on Vercel

## Product Goals
- Stand out from generic developer portfolios
- Demonstrate technical creativity (the OS metaphor itself IS the demo)
- Quick path from landing to understanding capabilities
- Projects should feel interactive, not just screenshots

## What Would Advance This Product
- Interactive project demos (embed or iframe live previews)
- Terminal window that actually runs commands (about, skills, contact)
- Desktop icons that launch project windows with real functionality
- Start menu with resume, links, contact info
- Easter eggs that reward exploration
- Blog/writing section as a text editor window
